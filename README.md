# Frontend Challenges

A collection of small frontend challenges built with **React** and **Tailwind CSS** using **Next.js**.  
These mini-projects are for practising frontend skills, including React components, state management, and UI styling.

### 1. Counter
A customizable counter with features like:  
- Increment and decrement  
- Step input to control the increment/decrement value  
- Reset button  

### 2. Guess the Number
A number guessing game with features like:  
- User inputs a guess between 0 and 100  
- Feedback if the guess is too low, too high, or correct  
- Reset functionality to start a new game  

### 3. Infinite Scroll
An infinite scroll demo using the [DummyJSON API](https://dummyjson.com/posts).  
- Fetches posts with **Axios** and pagination  
- Uses **Intersection Observer** to automatically load more posts when scrolling  
- Cancel tokens ensure unnecessary requests are cancelled when fetching new pages  
- Displays both **title** and **body** for each post
